新的博客

题图是前几天翻到的 iPod Shuffle

最早的时候应该是 2005 年左右，在 msn 的 space 上面写博客，后来跑去了 163 博客，在之后2009年的时候学了点 Linux 基础知识，就从 Wordpress 开始捣鼓起来没想到一直写到了 2017 年。

2017 年的时候生活变化比较多，当了爹，职业生涯也有了一些变化。突然追求的事情好像也有点不同了，并且又一次开始拍照。这次想的多了一些，没想到断断续续的拍到了2021年。但表达欲可能被摄影消耗掉比较多，就把博客关了。后来自己还是忍不住，断断续续在苹果的记事本上也写了几个月的日记。

这两年世界变化真的是很大，但我不是什么观察家，只是想留个地方继续随便碎碎念记录点东西。虽然说是 Github 吧，但偶尔写点日常吐槽的东西似乎也没啥。心态最大的不同就是懒得折腾博客皮肤之类的表面工作了，稳定就行，虽然 jekyll 一般来说是首选，毕竟用的人多。但再怎么说我也是一个 JavaScript 搬砖工，还是选了以前也用过的 Hexo 来继续折腾折腾。
